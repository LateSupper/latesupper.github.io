Vidilly

HTML <video> UI

Vidilly was created as a fun mini project, and in return provides a simple, plug-and-play video user interface.

Video source: W3Schools.com and Big Buck Bunny
Download

Usage

Option 1: Download

  1. Download vidilly.zip using the button above, then extract the vidilly folder to your codebase.
  2. Link the vidilly.css stylesheet to your web page in the <head> tag.
  3. Provide a <script> tag for vidilly.js in your web page at the end of the <body> tag.

Example:

<html>
    <head>
        ...
        <!-- "**" being the destination in your codebase -->
        <link href="**/vidilly/vidilly.css" rel="stylesheet">
    </head>
    <body>
        ...
        <!-- "**" being the destination in your codebase -->
        <script src="**/vidilly/vidilly.js">
    </body>
</html>

  1. Link the vidilly.css stylesheet to your web page in the <head> tag using the code provided below.
